# with umi example

## How to use

Download the example:

```
curl https://codeload.github.com/rsuite/rsuite/tar.gz/main | tar -xz --strip=2 rsuite-main/examples/with-umi
cd with-umi
```

Install it and run:

```
npm install
npm run dev
```

Open [http://localhost:8000](http://localhost:8000) to view it in the browser.
